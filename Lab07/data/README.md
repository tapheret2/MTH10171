# Data

Đặt file **Loan Approval Prediction Dataset** tại thư mục này và đổi tên thành:

```text
loan_approval_dataset.csv
```

Bộ dữ liệu thường có các cột như:

- `loan_id`
- `no_of_dependents`
- `education`
- `self_employed`
- `income_annum`
- `loan_amount`
- `loan_term`
- `cibil_score`
- `residential_assets_value`
- `commercial_assets_value`
- `luxury_assets_value`
- `bank_asset_value`
- `loan_status`

Biến mục tiêu của bài toán là `loan_status`, gồm hai lớp:

- `Approved`
- `Rejected`
