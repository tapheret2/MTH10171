## 🧪 Labs Overview

### Lab 01 – Các kiểu dữ liệu
Khám phá và phân biệt 3 loại dữ liệu:
- **Structured** (CSV/Excel): Dữ liệu bảng có cấu trúc rõ ràng
- **Semi-structured** (JSON): Dữ liệu bán cấu trúc
- **Unstructured** (TXT): Văn bản thuần túy

### Lab 03 – Xử lý dữ liệu với Pandas
Thực hành các thao tác xử lý dữ liệu:
- Đọc, lọc, sắp xếp và gộp DataFrame
- Phân tích dữ liệu bán hàng theo quý
- Bài tập HW1 tổng hợp

### Lab 04 – Trực quan hóa dữ liệu
Thực hành vẽ biểu đồ với Matplotlib / Seaborn:
- Biểu đồ cột, đường, tròn
- Phân tích và trình bày dữ liệu trực quan
